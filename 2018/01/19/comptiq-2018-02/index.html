さっきぃ提督の艦これラジオ、お待ちしております( ◜◡◝ )

インタビューの方は、やっぱり他の東山さん、野水さんと比べると提督目線の内容が多かったような気がしますｗ
あと、藤田さんってリアルイベントで提督たちと決起の鬨の声上げるのほんとに好きなんだなあと思いました( ◜◡◝ )
確かに、リアルイベントで他の提督はもちろん藤田さんと一緒に「頑張るぞー！」ってやると、俄然やる気になりますもんね。
前回の2017秋イベも、観艦式での涼月発表と決起があったからこそ頑張れたもんなー。
なので、ぜひイベントの前には藤田さんを筆頭にした決起集会を定期的に開いていただきたくですね( ◜◡◝ )

あと気になったのは、「今後やりたいこと」という質問に対して
