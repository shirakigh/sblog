キラキラ プリキュアアラモード 第15話 感想：あきゆか(のお付き合い)はじまってた。

2017-05-18
声優 / アニメ

こんばんは、しらきです。
プリアラ15話、観ました！！！！！
あきらさんの妹みくちゃん初登場にして10話以来の濃厚なあきゆかが楽しめる大満足の回でした( ◜◡◝ )( ◜◡◝ )( ◜◡◝ )
こっそりあおひまも凄かったし…あといちかのお姉さんっぷりも良かったしね( ◜◡◝ )

では今回も、ゆかりさまファンクラブ勢としてゆかりさんに偏りまくった感想を！

出会って早々に剣城姉妹の本質を見抜くゆかりさん

今回見ていて感じたのは、やっぱりゆかりさんて人を見る洞察力というか観察眼が凄いんだなーということ。
冒頭でみくちゃんが「いつもお姉ちゃんがお世話になってます！」って挨拶するシーン、中学生組は礼儀正しさに驚いてましたけど、ゆかりさんだけ何故かしたり顔で満足げに笑ってる姿が印象的で。
あれって、あの時点ですでにみくちゃんの中にあきらさんの「自分のこと二の次で誰かのために頑張っちゃう」って姿を見出して嬉しかったのかなーとか勝手に妄想してによによしています( ◜◡◝ )

さらに、みくちゃんがキラパティのお手伝いしたいって言い出したシーン。
みくちゃんに「今日はみくに任せてゆっくりしてて」って言われちゃうあきらさんを見て一人こっそり(でもないけど)笑っちゃってるゆかりさんな～～～～～～～～～～～～～( ◜◡◝ )( ◜◡◝ )( ◜◡◝ )
ラストであきらさんに「守られているのは妹さんじゃなくて、あきらの方かもね」って言ってますけど、なんとなく冒頭のこの時点でそこまで気づいてそうですごいなー、と。
でも気づいていても最後まで言わないのがまたゆかりさんらしくてもう～～～～～～！！！！！！！！！ってなります( ◜◡◝ )( ◜◡◝ )

手じゃなくて手首を取っちゃうゆかりさんの可愛さ( ◜◡◝ )( ◜◡◝ )( ◜◡◝ )

なんか今回、ゆかりさんすっごいあきらさんに対して優しくなかったですか？？？？？？
なんなの？？？？？10話を経てすでに彼女？？？？？彼女ヅラなの？？？？？？？
10話のラストで彼氏ヅラだったあきらさんに対して今回はゆかりさんが彼女ヅラなの？？？？なんなのあきゆかなの？！？？？そうなの！？？！？？！(興奮

いちかに「お二人はちょーっとお散歩してきてください」って言われて、いちかの意図を理解した上ではあるんだろうけど、嬉しそうにあきらさんの手を取っちゃうゆかりさんが可愛すぎてですね……
てか！！！！！あのシーン、ゆかりさんがあきらさんの手じゃなくて手首つかんでるとこくそ可愛くないですか？？？？？？？？？？？？
手じゃ駄目だったの？？？？照れなの？？？？？？それとも手首つなぎは主従関係の証なの？？？？？？「(私の)犬だから」とかそういうことなの？？？？？？？？？

あれ、あのまま引っ張っていって最終的にどうしたんですかねぇ…すぐに離したのか、それともそのままなのか、はたまた普通に手繋いじゃったりしたのか…！
個人的には、ゆかりさんがそろそろ離そうとした瞬間にあきらさんが普通に繋いできちゃって「あれ？」ってなるパティーンを希望します！！！！！！！！！！！！！！
あきらさんはそういうこと無意識にやってるといいと思う( ◜◡◝ )( ◜◡◝ )( ◜◡◝ )
あきらさんのそういうナチュラルな罪深さ希望( ◜◡◝ )

あ、ちなみに、「手首つなぎ」でググると色んな恋愛診断サイトがたくさん出てくるんですけど、その中の一つに「手首つなぎは甘えたい証拠」とか書いてあってしにました。

ゆかりさんたら…ゆかりさんったら…！！！！！！！！！！！！！(言葉にならない叫び

10話以降すっかりあきらさんのこと受け入れてるゆかりさんヤバイ

なんだか、今回見ていて、ゆかりさんたらもうすっかりあきらさんのことまるごと受け入れちゃってるんだな～～～～と思いました。
嵐を呼ぶ10話では、まだ考え方の違いから苛立ちを感じていたり、最終的には少しだけ歩み寄りを見せてましたけど。
今回ゆかりさんたらもうすっかりあきらさんのこと許しちゃってて受け入れちゃってて、なんだよゆかりさんちょろいな可愛すぎかよ！！！！！！！！！ってなりました( ◜◡◝ )

公園であきらさんに「みくちゃん、あきらにそっくりじゃない？」「お互いに、同じこと考えてるってこと」って言う時のゆかりさんの声の優しさやばくないですか…
いつからあきらさんに対してそんな優しい声出すようになっちゃったんですかゆかりさん…やっぱり彼女か、彼女なのかそうなんですかお付き合いはいつからですか…！！！！！！！！！！！！！

個人的に推しカプのターニングポイントを妄想するの大好き勢でして、10話で初めて名前呼びするようになったあきゆかがここに来て一気に関係が進展してる感じがしてソワソワしてました。
あきらさんはあんまり変わらないけど、ゆかりさんの変わりっぷりが凄すぎて…あきらさんすごい…っょぃ……

あ、でもあきらさんにも一つ気になったことが。

「みんなお姉ちゃんの言ってた通り！」

冒頭でのみくちゃんのこの台詞。
あきらさんがみくちゃんに対してキラパティの面々のことを話してたってことですけど、みくちゃんゆかりさんに対して「気まぐれでとーっても美人なゆかりさん」って言ってましたよね？？？？？？
「お姉ちゃんの言ってた通り」ってことは、あきらさんがゆかりさんについてみくちゃにそういう説明をしたってことですよね？？？？？？

…ぐっ、あきらさんがどんな雰囲気でどんな声でみくちゃんにそう言ったのか気になって仕方ない…！！！！！
多分最初にいちかのこと伝えて、ひまりちゃんの紹介して、あおちゃんのこと紹介して、最後にゆかりさんだよね、きっと。
どういうテンションで伝えたのかなあ…この一瞬の妄想だけで夢が広がる……( ◜◡◝ )
しかもさあ、それに対するゆかりさんの「正解」って返しがまた( ◜◡◝ )

あきゆかは…成長している…！！！！！！！！！！！

いちかの「キラっとひらめいた」の汎用性の高さは異常

今回、あきらさん回ではありましたけど、いちかのお姉さんっぷりがすごく良かったなあと思いました。
いちかの「キラッとひらいめいた！」は、基本スイーツに対して誰かの失敗を補うデコレーションのことですけど、今回はみくちゃんのフォローとしてキラッとひらめいてましたね。
「お姉ちゃんのために」と頑張るけれどうまくいかないと落ち込むみくちゃんに対して、外に連れ出したりスイーツ作りを提案したり。

いちかって誰かのマイナスの感情を拾うアンテナが敏感なのかなあ。
しかもフォローの仕方が嫌味じゃないから凄いというか。
言葉で励ますんじゃなくて、間接的に相手が元気になるように振る舞うって言う、いちかさんのフォローの仕方ほんとスマートで…いちかさんマジいちかさん…ってなる。

もしかしたらいちかこそ、「自分のことは二の次で他人のために頑張れちゃう」人なのかもですね。
凄いなあ、いちかさん。なんという主人公…！！！！！
なのにリオくんの怪しさには全く気が付かないって言うねｗ
もしかして単にリオくんにそれほど興味がないだけなんじゃ…

リオくんに対しては今回さすがのあきらさんもちょっと不信感を持っていたみたいでしたね。
もしかして来週のゆかリオ回への伏線だったりするのかな？？？？
あきゆかリオの三つ巴来ちゃうの？？？？来ちゃうの？？！？？？！？！？！！

来週はついにゆかりさん回！！！！！！！！！！！！！！！！！！！！！！！！！！

そう、来週は！！！！！！！！！！！！ついに待望のゆかりさん回ですよ！！！！！！！！！！！！！！！！！

キラキラ☆プリキュアアラモード 第16話予告 「キケンな急接近！ゆかりとリオ！」
https://youtu.be/JBiilIHlybI

こーーーーーーーれーーーーーーーーーなーーーーーーーー！！！！！！！！！！！！！！！！

この予告、ラストのゆかりさんの横顔が美しすぎてうっとりしてしまいます。
なんだろう、何を思っているんだろうこのゆかりさん…めっちゃ気になる……
これ坂の上からいちご坂の町並みを見下ろしているんだよね？
自分の住む町並みを眺めながら何を思うんだゆかりさん…！！！！！！！！！！！！！！！！

あきらさんは最近引っ越してきたって設定ですけど、ゆかりさんはずっとこの街で生まれ育ったのかな。
故郷や家に対する思い入れがありそうでなさそうでやっぱり強くありそうなゆかりさんですけど、次回少しでもその辺に触れてくれたら私はもう…もう…！！！！！
ここ最近ゆかりさんの幼少期妄想が爆発してるので是非ゆかりさんの幼少期について少しでも触れてくれたらと思ってるんですけどどうなのかな。
5話のおばあさんが再登場するみたいだし、期待しちゃっても良いのかな。

でも予告の文章で、

引用元：http://www.toei-anim.co.jp/tv/precure/

ゆかりの家でお茶会が開かれることになった。

ってあるから、多少なり琴爪家の事情が明かされそうですよね。
「お茶会」ってやっぱりそれなりのお家なんだろうなあ、きっと…「お茶会」とか「パーティー」とかあおちゃんが凄い嫌な顔しそうなワードだｗ

あとリオくんに関しては

密かに怪しい存在だと警戒している

ってあるから、やっぱりずっと警戒していたんだなあ。
てか、予告の第一声が「気に入らないわ！」で笑ったｗ
でもゆかりさんのこの怒りが、単にリオくん自身が気に入らないってんじゃなくて、リオくんが自分の「面白い」仲間たちに害がありそうで、だから「気に入らない」って憤ってるあたりすごくゆかりさんですごく好きです。

あきらさんやいちかとはまた違うベクトルで、ゆかりさんも誰かのために行動してるよね。
でもそれを「誰かのために」とは言わずに、「(私が)気に入らない」って、あたかも自分のためにみたいに言っちゃうゆかりさんほんと素敵だと思うんですよ……

ああ、次回楽しみだなあ…ほんと楽しみです。
残念ながら外出予定なのでリアタイはできないんですけども…っ！！！！！！！！！！！！！！！！(号泣
帰宅したら心を落ち着けて、深呼吸して視聴に臨みたいと思います。
あ、もちろん正座してｗ

あ、今回書けなかったけど、15話はあおひま的にもたくさん美味しいシーンがあったので、書けたらそのことも書きたいなあと思ってます。
プリアラ勢みんな魅力的なので、回を追うごとに思い入れが深くなって感想も長くなってしまう；；；；

それでは、また( ◜◡◝ )
